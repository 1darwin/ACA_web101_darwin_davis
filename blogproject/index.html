Explain how a browser determines what elements match a CSS selector.

October 1st, 2019

The browser reads each CSS file from top to bottom and the last element read is the one selected. If there are multiple CSS files and an element appears in multiple files, the one that is last in the last html file is the one that is used for formatting.

by Darwin Davis

What's the difference between element and tag?

October 1st, 2019

An element is a set of opening and closing tags. Tags are labels that are marked up at the beginning and the end of an element. All tags begin with a less-than sign, <, and end with a greater-than sign >, the only difference between an opening tag and a closing tag is the forward slash /.

by Darwin Davis

What do you think the "Cascade" is? How do you explain it?

October 1st, 2019

Cascading means that styles can fall (or cascade) from one style sheet to another, enabling multiple style sheets to be used on one HTML document.

by Darwin Davis

Explain the 3 ways to use CSS to style an html file.

October 1st, 2019

3 ways to use CSS to style an html file. Inline on the element in the HTML file, a style section in the HTML file, or a separate CSS file that is referenced in the HTML file.

by Darwin Davis

October 1st, 2019

It’s like putting a suit (CSS) on someone who just has on underwear and a t-shirt (HTML). HTML only allows for a vanilla website while adding CSS allow the website to be dressed up with all the bells and whistles.

by Darwin Davis

How do you choose the correct CSS selector?

October 1st, 2019

You must match the CSS selector to the HTML element. The format in the CSS must be the appropriate format to select the correct type of element.
